# Ошибки параметров

`MotionParamError.code` — стабильный машинный идентификатор причины. Ошибки
движка содержат в `message` только тот же код: входные значения не отражаются
в runtime-строку. Совместимый внешний конструктор сохраняет переданный текст и
получает `LM000`. Причина и исправление находятся в каталоге; ветвление строится
по `code`.

Коды со статусом `retired` не переиспользуются. `LM000` обозначает ошибку,
созданную внешним кодом через совместимый строковый конструктор.
