# Бенчмарк @labpics/motion — размер и скорость

> Канонический файл бенчмарка, единственный источник правды.
> Английский `docs/benchmark.md` удалён, потому что две версии одного
> документа неизбежно расходятся — а расхождение в цифрах хуже их отсутствия.

## Статус честности

| Метрика | Статус |
|---|---|
| Размер наших бандлов | **Измерено**, воспроизводимо: `pnpm build && pnpm size` |
| Размеры vendor'ов | **Заявлено vendor'ом, нами не измерено** (motion.dev 2026-07, bundlephobia) |
| Скорость / runtime | **Достоверных сравнительных чисел пока нет** — см. раздел «Скорость» |

Прежняя таблица скорости («94% визуального прогресса при freeze», «ретаргет
0.5–0.8 vs 2.1» и т.п.) удалена: эти числа происходили из симуляций и ожиданий
линии PR#77 и не выводились из кода бенчмарка. Публиковать их как измерения —
значит выдавать желаемое за действительное; в этом файле остаётся только то,
что воспроизводится командой.

## Сравнение с конкурентами (размер)

Колонки vendor'ов — **заявлено vendor'ом, нами не измерено**: источники
motion.dev (снапшот 2026-07) и bundlephobia. Мы не прогоняли их сборки через
наш гейт, поэтому сравнение честно только по порядку величины.

| Фича | @labpics/motion (измерено) | Motion (vendor) | GSAP (vendor) | anime.js (vendor) |
|---|---|---|---|---|
| ядро + пружина | 2.16 KB gz (core) + 1.66 KB (spring) | 2.6 KB (mini) | ~23–26.6 KB core | ~12 KB |
| stagger | 0.74 KB gz | bundled | bundled | bundled |
| timeline | 1.84 KB gz | — | bundled | bundled |
| animate (one-liner) | 10.24 KB gz / 10968 B import-cost | 2.6 KB mini / 18 KB full | ~23+ KB | ~11–12 KB |
| compositor | 6.27 KB gz | hybrid | main-thread | WAAPI частично |

Субпуть-изоляция (`sideEffects: false`) гарантирует: не импортируешь модуль —
не платишь за него ни байта (проверяется сценариями import-cost в гейте).

## Скорость: что известно честно

- Сравнительного runtime-бенчмарка с достоверными числами **нет**. Каркас
  `bench/compare/bench.mjs` (Playwright + CDP: CPU-throttle, main-thread
  freeze, ретаргет под нагрузкой) существует, но его прогоны из линии PR#77
  невоспроизводимы — честный прогон вынесен отдельной задачей в BACKLOG.md.
- Архитектурные факты (дизайн, а не измерения): closed-form пружина без
  пошагового интегрирования, C¹-непрерывный ретаргет, compositor с WAAPI
  handoff для анимации вне main-thread. Мы *ожидаем* преимущество в сценариях
  freeze/throttle, но опубликуем числа только после честного прогона.
